# ストアフロントウィジェット

[!DNL Live Search]をインストールすると、ストアフロントに2つの新しいウィジェットが表示されます。

- [!DNL Live Search] [ ポップオーバーウィジェット ](storefront-popover.md)は、検索結果を含む検索フィールドの下に表示されるボックスです。
- [製品リストページ ウィジェット ](plp-styling.md) （PLP）は、ファセットと同義語のサポートを備えた検索可能な製品リストページを提供します。 ウィジェットがインストールされ、Live Search 4.0.0以降で有効になります。

自社のスタイルやブランドガイドラインに合わせて、これらのウィジェットのルックアンドフィールをカスタマイズできます。

## ライブサーチのポップオーバーウィジェット

Commerce ストアフロントの検索ボックスに入力を開始すると、[!DNL Live Search]さんが[ ポップオーバーウィジェット ](storefront-popover.md)で、おすすめ商品と上位の検索結果のサムネイル画像を返します。

![[!DNL Live Search popover]](assets/storefront-search-as-you-type.png)

ポップオーバーウィジェットと、ストアフロントのウィジェットをカスタマイズする方法について詳しくは、[[!DNL Storefront Popover]](storefront-popover.md)を参照してください。

## 製品リストページウィジェット

ストアフロントのポップオーバーからすべての結果を表示するには、をクリックすると、製品リストページウィジェットに、検索を継続的に調整するために使用できる機能が表示されます。

![製品リストページのウィジェット結果](assets/plp-css-widgets.png)

製品リストページウィジェットとストアフロントのウィジェットをカスタマイズする方法について詳しくは、[製品リストページウィジェット ](plp-styling.md)を参照してください。
