# マーチャンダイジングルールのベストプラクティス

コンバージョンと収益を最適化するには、効果的な&#x200B;**検索ルール**、強力な&#x200B;**デフォルトリスト** ルール、および&#x200B;**[カテゴリルール](add.md#category-rules)** （ベータ版）を実装します。 販売データ、在庫、プロモーション、[&#x200B; インテリジェントランキング &#x200B;](add.md#intelligent-ranking)を使用してランキングを調整します。

よく考えられた&#x200B;**デフォルトのルール**&#x200B;を確立することが重要です。 [既定のルール &#x200B;](overview.md#default-rule)は、より具体的な検索ルールが適用されないときに検索結果を最初に並べ替える方法を決定します。これにより、見つけやすさと購入の可能性が向上します。 定期的に見直して、買い物客のニーズや施策に対応できるようにしましょう。

## 検索ルールを最適化するためのヒント

- 販売量の多い商品や最近の販売活動をピン留めしたり強化したりできます。
- 高い評価と好意的なレビューを得た商品に優先順位を付ける。
- 在庫商品が上位に表示されていることを確認する。
- 関連性を損なうことなく、利益率の高い商品をわずかに優先する。
- セール中の商品や特別プロモーションの一部をハイライト表示します。
- プロモーション期間中の日付範囲を使用して、プロモーション期間中または販売期間中に検索ルールを自動的に設定できます。
- 「おすすめ」や「最も閲覧されたもの」など、[&#x200B; インテリジェントなランキング &#x200B;](add.md#intelligent-ranking)を使用して、個々の買い物客の行動に基づいて検索結果をカスタマイズします。
- インテリジェントランキング戦略が異なるクエリの実際の検索結果にどのような影響を与えるかをプレビューするには、常に「ルールをテスト」パネルを使用します。

## カテゴリルールのヒント

>[!IMPORTANT]
>
>カテゴリルールはベータ版です。

- 季節的なコレクションやおすすめの部門など、厳選された注文が検索と同じくらい重要な高トラフィックまたはマージンの[&#x200B; カテゴリページ &#x200B;](add.md#category-rules)で&#x200B;**カテゴリルール**&#x200B;を使用します。
- **インテリジェントランキング** （例：トレンド、最も閲覧されたカテゴリ）を、買い物客がそのカテゴリを閲覧する方法に合わせます。カテゴリーページでは、検索ルールと同じように検索クエリテキストが使用されません。 [&#x200B; インテリジェントランキング &#x200B;](add.md#intelligent-ranking)を参照してください。
- **ピン**、**ブースト**、**ベリー**&#x200B;をキャンペーンプランと一貫して適用します。通常、手動配置は、買い物客がリストに&#x200B;**デフォルトの並べ替え**&#x200B;を使用している場合にのみ適用されます。 [手動ランキング &#x200B;](add.md#manual-ranking)を参照してください。
- エディターの&#x200B;**カテゴリ** ルールフローでプレビューし、公開後にストアフロントで検証します。これは、検索時の「ルールをテスト」パネルで使用するのと同じ規律です。
