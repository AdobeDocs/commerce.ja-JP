# ストアフロントの設定

AEM Assets統合では、従来のCommerceでホストされる画像ではなく、AEM Assetsで管理される製品画像が表示されます。 この統合により、Adobeのコンテンツ配信ネットワーク（CDN）を介して、高度な最適化、切り抜き、配信などの強化画像管理機能が提供されます。

Edge Delivery Servicesを搭載したCommerce Storefront でAEM Assetsを使用するには、「[EDS AEM Assets設定 &#x200B;](https://experienceleague.adobe.com/developer/commerce/storefront/setup/configuration/aem-assets-configuration/)」トピックで説明されているストアフロント設定を行います。
